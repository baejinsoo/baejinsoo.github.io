### 백준(BOJ) - 캠핑

[백준(4796) - 캠핑](https://www.acmicpc.net/problem/4796)

##### 해결방법 

어려울게 없는 단순한 그리디 문제였다.

```python
import sys
input = sys.stdin.readline

cnt = 1
while 1:
    l, p, v = map(int, input().split())
    if l + p + v == 0:
        break

    res = (v // p) * l
    res += min((v % p), l)
    print('Case %d: %d' % (cnt, res))
    cnt += 1

```
