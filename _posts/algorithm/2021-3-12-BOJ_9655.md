### [백준(9655) - 돌게임](https://www.acmicpc.net/problem/9655)

> 문제풀이: 수학, 파이썬

문제를 풀고서 힐링을 느꼈다. 오랜만에 쉬운문제를 풀어보니 기분이 좋았다.

단순히 문제를 읽고 결과를 생각해보면 되는 쉬운 수학 문제이다.

```python
n = int(input())

if n % 2 == 1:
    print("SK")
else:
    print("CY")
```
